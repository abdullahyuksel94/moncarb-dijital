# MONCARB Dijital — Bilgi Tabanı

  Bu repo, MONCARB Cutting Tools dijital projesinde öğrenilen her şeyi belgeler.
  Yeni projelerde referans, yapay zeka eğitimi ve tekrar kullanım için tasarlanmıştır.

  ## Dosyalar

  | Dosya | İçerik |
  |---|---|
  | firma-bilgileri.md | Şirket bilgileri, iletişim, sertifika |
  | kategori-yapisi.md | Ürün kategorileri ve URL yapısı |
  | teknik-stack.md | Teknoloji yığını ve dosya yapısı |
  | ceviri-sistemi.md | TR/EN/RU çeviri sistemi, L() fonksiyonu |
  | urun-sayfasi-pattern.md | Ürün sayfa şablonu ve veri yapısı |
  | brosur-pdf-sistemi.md | FlipbookCatalog ve PDF yönetimi |
  | mobil-cozumler.md | Mobil uyumluluk düzeltmeleri |
  | onemli-kararlar.md | Mimari kararlar ve kurallar |

  ## Proje Özeti

  MONCARB Cutting Tools İstanbul merkezli bir karbür kesici takım üreticisidir.
  Bu web sitesi TR/EN/RU çoklu dil desteği olan bir kurumsal tanıtım sitesidir.

  > **Kural:** Fiyat bilgisi kesinlikle gösterilmez — sadece sipariş kodu ile iletişime yönlendirme yapılır.
